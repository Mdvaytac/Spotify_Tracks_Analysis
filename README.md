# Spotify SQL Analytics Project

This project contains advanced SQL analysis tasks created using Spotify music datasets (`artists` and `tracks` tables).

## Topics Covered
- Artist popularity analysis
- Song duration analysis
- Window functions
- Ranking functions
- Percentile calculations
- Running totals
- Music mood classification
- Year-over-year growth analysis
- Statistical outlier detection

## Tables Used
- `artists`
- `tracks`

## SQL Skills Practiced
- JOIN
- GROUP BY
- Subqueries
- CTEs
- ROW_NUMBER()
- DENSE_RANK()
- PERCENT_RANK()
- NTILE()
- LAG()
- Window Functions

## Purpose
This project was created for SQL practice, portfolio building, and improving analytical thinking using real-world music streaming data.
